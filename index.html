Dimple.js plot #1

Here some data regarding the Challenger, and different test launches, is presented.

Above, you can see damage index compared to the temperature for different launched. That graph has been constructed using D3.js.

Beside that we have the Erosion incidents compared to the temperature. This graph was made using Dimple.js and can be interacted with using the buttons underneath it.

And last but not least, in the bottom right corner we hade another plot made using Dimple.js where you can see what number of erosion incidents occured for different flights. The buttons below it can be used to tidy up the graph.
